💡 Key Findings — Phase 1
0.97

RF Classifies Current Aero State

From instantaneous features alone — confirming the threshold rule (Speed≥240 AND Gear≥6) is fully recoverable from telemetry. Sets the ceiling Phase 2 tries to match with 1s lookahead.

~5%

Anomalous Driving Windows

Isolation Forest flags ~5% of telemetry as anomalous. These cluster near braking zones, sector transitions, and traction limit events on the Suzuka GPS map.

k=4

Four Driving Zones

K-Means identifies: high-speed aero (X-Mode), medium cornering, heavy braking, and slow hairpin approach — matching known Suzuka sectors.

Limit

No Cross-Circuit Generalisation

Phase 1 trains and tests on the same race. This directly motivates Phase 2: do these patterns hold when tested on a circuit the model has never seen?

💡 Key Findings — Phase 2
0.963

LR Beats All Deep Models

The simplest model — instantaneous LR — is best overall. Complex temporal memory backfires: models memorise circuit-specific speed levels rather than learning general physics.

0.728

GRU Collapses at Monaco

GRU AUC drops to 0.728 at Monaco. F1-xmode = 0.005 — never predicts X-Mode. Transformer achieves 0.956 on the same circuit and same data.

0.956

Transformer Most Robust

Attention-based model maintains consistent cross-circuit performance by relying on instantaneous state features (Throttle, Speed) rather than circuit-calibrated dynamics.

H=25

2.5s Horizon: All Fail

AUC-ROC below 0.79 at H=25 for every model. At H=50, all near-random. Practical systems should operate within ~1 second look-ahead.

−0.27

Transitions Are Hardest

Within ±0.5s of each actual mode change, F1 drops by ~0.27 mean. Fundamental task property — the model predicts a future state the telemetry has not yet announced.

Root

Circuit Memorisation

Models encode absolute speed levels per circuit. Monaco's lower speed envelope means the "fast straight" signal never fires for GRU/RF-lag — Phase 1 found the anomalies; Phase 2 reveals their structural cause.
